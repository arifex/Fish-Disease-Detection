# Fish-Disease-Detection
Real-Time Image Based Fish Disease Detection with Treatment Recommendation Application

**Abstract:**
To avoid the economic losses it's necessary to detect fish diseases in the early stage. It also assists in minimizing unwarranted chemicals. But manual diagnosis is slow, and requires the availability of experts. This paper presents an image-based real-time fish disease detection system that is capable of classifying seven most widespread conditions in freshwater. Among these six are diseases and another one is healthy fish. This identifies the disease with a handsome accuracy and also provides basic treatment recommendations. The system employs transfer learning with an InceptionV3-based convolutional neural network which is publicly available Bangladeshi fish disease image dataset. Proposed model has a classification accuracy of about 97%. All these predictions are fed into a real-time application that receives images of fish captured by the user. It images the picture and gives the estimated disease category with likelihood figures, and provides brief literature-based treatment advice to the farmers. The result shows us it's possible to help farmers by predicting the diseases and treatment recommendations.
